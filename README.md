# Superstore Data Analysis

#  Project Overview
This project analyzes the Superstore dataset using Python. It includes data cleaning, exploratory data analysis, and visualization to extract meaningful business insights.

# Tools Used
- Python
- Pandas
- Matplotlib / Seaborn

# Analysis Performed
- Cleaned dataset (handled missing values and duplicates)
- Calculated total sales
- Identified top 5 products by sales
- Analyzed monthly sales trends

# Key Insights
- Sales show variation across different months
- Few products contribute to major revenue
- Seasonal trends are visible in sales data

# Visualizations
- Monthly Sales Trend
- Top 5 Products by Sales
- Sales by Category


# Conclusion
This project demonstrates basic data analysis skills and helps in understanding business insights from raw data.
